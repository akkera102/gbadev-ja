# gbadev-ja

GBA開発Wikiは、GameBoy Advanceについての技術資料を個人的に収集しているページです。
http://akkera102.sakura.ne.jp/gbadev/


## How to Use

- [Doc.1 開発環境のインストール](http://akkera102.sakura.ne.jp/gbadev/?doc.1)をご覧ください。

## How to Compile

- リポジトリをダウンロードした後、make_all.batを実行します。各フォルダ内にサンプルROMが生成されます。
